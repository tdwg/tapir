Resource Directory (RDDL) for TAPIR 1.0

Overview

This is TAPIR version 1.0. For more information on TAPIR see any of the following resources listed here.

Note: While this document is human-readable, RDDL-aware processors may use it to automatically retrieve resources such as different schema versions for validating documents. See http://www.rddl.org for further information.

The root (target) namespace of TAPIR version 1.0 is: http://rs.tdwg.org/tapir/1.0.
